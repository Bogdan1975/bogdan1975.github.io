Slider styling using "styled" directive.

Now right way to styling slider component is using "styled" directive. Below is example with using "styleBlock" attribute of "styled" directive.

The styles defined by "styleBlock" will overwrite styles defined by skin ("default" in case of 'skin' attribute not setted). If you don't want to have skins style, you must set "skin" to "none" value.

Slider styling using embeded skins.

The easiest way to styling slider component is using embeded skins. In this version (1.0.2) component have skins: "default", "simple-1", "elegance-1", "elegance-2", "rear-sight"

"skin" attribute processed by "styled" directive, but applying of this directive is not required - if "skin" is setted, "styled" will be applied automaticaly
